# IPL-Performance--Analytics-Dashboard
Interactive IPL Performance Analytics Dashboard built using Power BI

## Overview
An interactive Power BI dashboard for analyzing IPL player and team performance.

## Features
- KPI Cards (Matches, Runs, Fours, Sixes, Wickets)
- Top 10 Batsmen
- Top 10 Bowlers
- Team-wise Run Analysis
- Venue-wise Match Analysis
- Interactive Filters

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV
